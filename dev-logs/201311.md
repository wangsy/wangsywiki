ADT (Android Developer Tools)
------------------------------
ubuntu 13.10 에서 ADT(Android Developer Tools)를 실행시키기 위해서는 다음과 같은 작업이 필요하다. 64bit Ubuntu일 경우 필요하다고 한다.

```
$ sudo apt-get install lib32stdc++6
```
