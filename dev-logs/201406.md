* Windows 8 에서 git-client 설치 원함.
 * [msysgit 설치](http://msysgit.github.io/)
  * 실행하면, bash 가 뜬다.
  * `~/.ssh/id_dsa` 복사해 넣는다.
  * `ssh-agent bash` 를 실행한 다음, `ssh-add ~/.ssh/id_dsa` 해 준다.
  * 그 다음부터는 비밀번호를 넣지 않고도 git command 를 계속 쓸 수 있다.
